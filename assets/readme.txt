Wiremonkey Internet Connection Tracking is a easy to use plugin,helps your site users to keep track of their  internet connection status while they are on your website.

== Description ==
Wiremonkey is a responsive and highly compatible wordpress plugin with minimal user friendly configurations.This plugin is integrated with WiremonkeyJS javascript plugin from galactas to achieve internet connection status checking .

== Installation ==
Manual installation
1. Download and unzip the wiremonkey-internet-connection-tracker.zip
2. Upload unzipped files into the `/wp-content/plugins/` directory trough web file manager or ftp/sftp.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Find the configuration menu under the "Settings" Menu.
